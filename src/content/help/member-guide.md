# Saltus 特進 使い方ガイド

このページでは、個別MTG知見ストック＆サポート管理システムの基本的な使い方を説明します。  
※画面キャプチャは実際の操作画面です。

> **ヒント:** 不明点は右下の青い吹き出しボタンから問い合わせできます。

---

## はじめに

本システムでは、次の操作ができます。

- **MTG議事録** — 1on1 MTG の内容・宿題事項の記録
- **問い合わせ** — 使い方・システムに関する質問
- **設定** — パスワード変更

![ログイン後のダッシュボード全体像](/help/02-dashboard.png)

| 権限 | 主なできること |
|------|----------------|
| メンバー | 自分の MTG・問い合わせのみ |
| マネージャー | 全員の MTG 閲覧、自分の問い合わせ |
| システム管理者 | 全問い合わせ閲覧、ユーザー権限管理 |

---

## ログイン・新規登録

1. 自身で設定された **メールアドレス** と **パスワード** でログインします。
2. 初回のみ **新規登録** からアカウントを作成できます（権限は自動で「メンバー」）。

![ログイン画面](/help/01-login.png)

- パスワードを忘れた場合は「パスワードをお忘れの方」から再設定できます。
- マネージャー・管理者権限への変更はシステム管理者へ依頼してください。

---

## MTG議事録

### 一覧・検索

サイドバーの **MTG議事録** から、記録一覧を確認できます。

- キーワード・実施日で検索が可能
- マネージャー・管理者は **投稿メンバー** で絞り込み可能

![MTG議事録一覧](/help/03-mtg-list.png)

### 新規投稿

1. **＋ 新規投稿** をクリック
2. **MTG実施日**・**会話内容**・**宿題事項** を入力
3. **保存** をクリック

![MTG投稿画面](/help/04-mtg-new.png)

- 会話内容は Markdown 形式（見出し・箇条書き・太字）に対応しています。

### 詳細・編集

一覧のカードをクリックすると詳細画面が開きます。

- **宿題事項** — チェックを入れると完了（取り消し線）
- **コメント** — MTG に関するやり取り（コメント自体は編集不可）
- **編集** ボタン → 内容を修正 → **完了** で保存

![MTG詳細画面](/help/05-mtg-detail.png)

## 問い合わせ

### 新規起票（右下の青ボタン）

画面右下の **吹き出しボタン** から、いつでも問い合わせできます。

- **システムへの問い合わせ** — 不具合・改善要望
- **使い方の問い合わせ** — 操作でわからないこと

![問い合わせモーダル](/help/06-support-fab.png)

### 一覧・スレッド

**問い合わせ一覧** から過去の問い合わせを確認し、スレッド形式で返信できます。

- メンバー・マネージャー — 自分が起票したもののみ
- システム管理者 — 全ユーザーの問い合わせを確認・返信

![問い合わせスレッド](/help/07-support-thread.png)

## よくある質問

**Q. MTG を投稿したのに一覧に出ません**  
A. ログイン中のアカウントで投稿したか確認してください。メンバーは自分の投稿のみ表示されます。

**Q. 他人の MTG が見えません**  
A. メンバー権限では正常な動作です。マネージャー以上は全員分を閲覧できます。

**Q. 問い合わせのステータスを変えたい**  
A. 起票者は「取り下げ」のみ可能です。それ以外はシステム管理者が変更します。

1. 本番画面のスクリーンショットを撮影

3. GitHub に push → Render が自動再デプロイ

ファイル名の例: `01-login.png`, `02-dashboard.png`, … `08-settings.png`
